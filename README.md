A simple and modern movie search application built using **vanilla HTML, CSS, and JavaScript**.  
Search for any movie and instantly get results with posters and release years using the **OMDb API**.

## Features
- Search movies by name
- Clean, cinematic UI
- Dynamic movie cards
- Handles missing posters gracefully
- Keyboard (Enter key) support

## Tech Stack
- HTML5
- CSS3 (Flexbox)
- JavaScript (Fetch API)
- OMDb API
